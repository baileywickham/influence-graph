# Influence Graph
Studying influcence on twitter through a graph theoretic perspective.

## Requirements
- [python-twitter](https://python-twitter.readthedocs.io/en/latest/)
- twitter authentication file with one string per line, in order:
    - api key
    - api secret key
    - access token
    - access token secret
